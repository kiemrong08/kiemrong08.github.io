• Đầu tiên là cài đặt Python 3 và Python 2
  • Đặt đường dẫn của Python 3 vào PATH của Windows (thiết lập môi trường chạy)
  • Sau đó cài pip (Sử dụng command line)
  • Cài virtualenv bằng pip
  • Sau khi cài đặt xong xuôi, tạo thư mục djangogirls ở nơi bất kỳ trong ổ cứng ví dụ D:\djangogirls
  • Di chuyển command line đến thư mục vừa tạo, sau đó tạo môi trường làm việc với python3 bằng lệnh virtualenv ENV VD: C:\Users\Name\djangogirls>virtualenv myvenv. Trong đó myvenv chính là tên của môi trường ảo của bạn.
  • Chạy môi trường ảo bằng lệnh : C:\Users\Name\djangogirls> myvenv\Scripts\activate
  • Bạn sẽ thấy bạn đang ở trong môi trường ảo: (myvenv) C:\Users\Name\djangogirls>
  • Bắt đầu cài đặt Django, cài đặt pip đầu tiên bằng lệnh (myvenv) ~$ pip install django==1.8
  • Mục đích là tách biệt các gói cài đặt cũng như các ứng dụng khác đã cài qua pip.
